69 Get python-sutime from Git, cd into the directory, then run maven:
70 ``` bash
71 >> git clone https://github.com/FraBle/python-sutime.git
72 >> cd python-sutime
73 >> mvn dependency:copy-dependencies -DoutputDirectory=./jars
74 ```
75 Then copy the sutime and jars folder into our task6 directory.
76 
77 If you get a java.lang.RuntimeException: Class edu.stanford.nlp.python.SUTimeWrapper not found error, then jpype is
78 pointing to the wrong Java JDK library. Delete all except 1.8 and it should run.
