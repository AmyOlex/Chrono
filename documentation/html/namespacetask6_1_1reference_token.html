Function to convert a list of tokens into a list of refToken objects.

Author
Amy Olex
Parameters
tok_listThe list of tokens (required)
id_counterThe number the ID counter should start at. Default is 0.
spanA list of spans for each token in the tok_list. Must be the same length as tok_list. Assumes it is a one-to-one relationship in the same order as tok_list.
posA list of part-of-speech tags for each token in the tok_list. list. Must be the same length as tok_list. Assumes it is a one-to-one relationship in the same order as tok_list.
temporalA boolean list of 0's and 1' indicating which token contains temporal information. Must be the same length as tok_list. Assumes it is a one-to-one relationship in the same order as tok_list.
remove_stopwordsA boolean that, if true, removes tokens in the stopword list. Defaults to False.
Returns
A list of refToken objects in the same order as the input tok_list.
